# روافدكم

هذا مشروع Flutter مبدئي يستند إلى PRD المرفق، مع branding جديد باسم "روافدكم".

## التشغيل

1. تثبيت Flutter SDK.
2. تشغيل:
   flutter pub get
   flutter run
